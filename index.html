VPS（DigitalOcean）搭建 Shdowsock服务

发表于 2018-01-16

VPS（DigitalOcean）搭建 Shdowsock服务

1 注册DIgitalOcean账号

Digital Ocean官网注册。注册时需要绑定“VISA信用卡”或者“Paypal账户（可以绑定银联的借记卡）”。

  • 绑定信用卡的方式，会从账户中扣除1刀，然后很快返还（确认账户可以用于后续付款）
  • 绑定Paypal的方式需要首先至少充值5刀
  • 直接点击我的推广链接在注册后可以得到10刀赠送（使用Paypal注册的用户仍然需要至少充值5刀）
  • 拥有学生邮箱的朋友们，首次使用github education package 可以获得50刀的优惠

2 创建 一个“Droplet”虚拟机

点击测试各地机房的速度和延迟,作者推荐使用旧金山机房

创建Droplet时，可以选择添加ssh keys ，不添加系统将会讲root密码发送到注册邮箱，首次登陆自己设定密码
添加SSH公钥，用于SSH远程连接主机。生成密钥的命令为 ssh-keygen -t rsa -C “yourmail”

3 在创建的服务器上安装shadowsocks服务

官方使用说明

终端使用DIgitalOcean网页版，或者Xshell，Putty等均可，使用ssh连接到创建的服务器

  • 使用

    1. 命令行配置使用

    2. 配置文件使用

      • 创建配置文件 /etc/shadowsocks.json

      • 配置文件的作用
        |名称|解释|
        |:—:|:–:|
        |server|监听服务器ip地址|
        |server_port|监听服务的端口号|
        |local_address|本地监听地址|
        |local_port|本地监听的端口号|
        |password|连接服务使用的认证密码|
        |timeout|连接超时时间|
        |method|加密方式，默认aes-256-cfb|
        |workers|仅仅在unix、linux下有用|

在win10使用pycharm远程开发

发表于 2018-01-15

win10使用pycharm远程开发

应用场景

使用pycharm(pro)连接远程python解释器，在本地编写程序，远程运行，并通过xming或者xmanager等软件返回图形数据

环境简介

  • 本地操作系统Windows10， ssh终端软件：xshell或者putty等，xserver软件：xming，xmanager
  • 远程操作系统Ubuntu16.04，开启ssh服务

软件准备

pycharm，拥有学生邮箱的朋友可以授权使用专业版一年，推荐支持正版（注：远程解释器只在专业版中支持）。 当然无法使用专业版的朋友可以在远程主机上安装pycharm 社区版，在终端配合xming远程开发（类似teamviewer的效果）

xshell个人使用免费， putty免费 ，二选一

xmanager收费，建议土豪使用，xming 免费， 推荐

配置

ssh client 开启 x11 forwarding 选项
pycharm config 设置 DISPALY 环境变量， 具体在远程终端 echo $DISPLAY 查看
